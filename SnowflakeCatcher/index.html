Snowflake Catcher

Note: Program may lag on some computers.

Catch the snowflakes!

Hold mouse: Draw on the screen.
e: Draw diagonal across the screen.
q: Set the first point to draw a line. Creates a red line as a guide for where the line will draw. Red line does not catch snowflakes. Red line can also be used as an eraser.
w: Set the second point for the line and draw it.
r: Clear drawings.
