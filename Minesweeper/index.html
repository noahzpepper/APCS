Minesweeper

A remake of the classic Windows game Minesweeper.

When you win or lose, press enter to return to the pause menu to start a new game.

Timer may run inaccurately on some computers.

Left click: Reveal a tile.
Right click: Mark a tile.
Center click: Reveal nearby tiles (if fully flagged).
Enter: Toggle the pause screen.
Spacebar: Change difficulty on pause screen, change theme on game screen.
a: Read more about the game (while on pause screen).
